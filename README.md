### Setup 
```
1. git clone https://github.com/Zevubu/pet-finder
2. cd pet-finder in your terminal
3. npm install
4. Modify credentials in connection.js
5. node server.js
6. Go to http://localhost:5050 or another port if you modified the code

```
### Requirements
```
1. MySQL Database
2. MapQuest API Key

```
